# 简历筛选助手 - Chrome 插件

一个帮助用户在招聘网站（如 BOSS 直聘）快速筛选简历的 Chrome 浏览器插件。

## 功能特性

### 🎯 核心功能
- **学历筛选**: 支持按学历要求筛选（大专/本科/硕士/博士）
- **工作经验筛选**: 按工作经验年限筛选
- **薪资范围筛选**: 设置薪资范围进行筛选
- **关键词筛选**: 
  - 关键词包含：简历必须包含所有指定关键词
  - 关键词排除：包含任一排除关键词的简历将被过滤
- **年龄筛选**: 按年龄范围筛选

### 🎨 交互功能
- **高亮匹配**: 符合条件的简历会以绿色边框高亮显示
- **隐藏不匹配**: 自动隐藏不符合条件的简历
- **统计信息**: 实时显示筛选统计（匹配/不匹配/总计）
- **一键清除**: 快速清除所有筛选条件

### ⚙️ 高级功能
- **预设管理**: 保存多套筛选规则预设，快速切换
- **配置导入/导出**: 支持配置的导入和导出
- **实时同步**: 使用 Chrome Storage API 同步配置

## 安装方法

### 开发者模式安装

1. **下载项目**
   ```bash
   git clone <repository-url>
   cd chrome_boss_plugin
   ```

2. **打开 Chrome 扩展程序页面**
   - 在 Chrome 浏览器地址栏输入: `chrome://extensions/`
   - 或者: 菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在扩展程序页面右上角，打开"开发者模式"开关

4. **加载插件**
   - 点击"加载已解压的扩展程序"
   - 选择项目根目录 (`chrome_boss_plugin`)
   - 插件安装完成

## 使用方法

### 基本使用

1. **打开支持的招聘网站**
   - BOSS 直聘 (zhipin.com)
   - 其他支持的招聘网站

2. **打开插件弹窗**
   - 点击浏览器工具栏中的插件图标

3. **设置筛选条件**
   - 选择学历要求
   - 设置工作经验
   - 输入关键词（包含/排除）
   - 点击"应用筛选"

4. **查看结果**
   - 符合条件的简历会以绿色边框高亮
   - 不符合条件的简历会被隐藏或标记
   - 页面顶部显示筛选统计信息

### 高级设置

1. **打开选项页面**
   - 在插件弹窗中点击"高级设置"
   - 或右键插件图标 → 选项

2. **配置详细规则**
   - 设置薪资范围
   - 设置年龄范围
   - 更多筛选选项

3. **管理预设**
   - 保存当前配置为预设
   - 加载已保存的预设
   - 删除不需要的预设

4. **导入/导出配置**
   - 导出配置为 JSON 文件
   - 从 JSON 文件导入配置
   - 重置为默认设置

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **Chrome Extension API**: Manifest V3
- **存储**: Chrome Storage API
- **UI**: 原生 JavaScript + CSS（无外部依赖）

详细开发规范请参考 [.cursorrules](.cursorrules) 文件。

## 支持的网站

- BOSS 直聘 (zhipin.com)
- 其他招聘网站（需要根据实际网站结构调整选择器）

## 注意事项

1. **选择器适配**: 不同招聘网站的 HTML 结构可能不同，需要根据实际情况调整 `content.js` 中的选择器

2. **性能优化**: 
   - 插件使用防抖机制避免频繁重新筛选
   - 监听页面变化，自动处理动态加载的内容

3. **隐私保护**: 
   - 所有数据存储在本地（Chrome Storage）
   - 不向外部服务器发送任何数据
   - 不收集用户隐私信息

## 常见问题

### Q: 筛选不生效？
A: 检查以下几点：
1. 确保已启用筛选（弹窗中的"启用筛选"开关）
2. 检查是否在支持的网站上使用
3. 查看浏览器控制台是否有错误信息

### Q: 如何自定义筛选规则？
A: 打开选项页面，在"高级设置"中可以配置详细的筛选规则。

### Q: 配置会丢失吗？
A: 不会。配置保存在 Chrome Storage 中，会同步到你的 Chrome 账号。
